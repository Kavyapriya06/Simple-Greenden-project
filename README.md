# Greenden – Responsive Plant Store Website

# Overview
    Greenden is a responsive and modern plant store website developed using HTML5 and Tailwind CSS. The website presents a curated collection of indoor and outdoor plants through a clean, intuitive, and visually appealing interface. Designed with a mobile-first approach, it delivers a seamless browsing experience across desktop, tablet, and mobile devices.

This project demonstrates responsive web design, structured page layouts, and the effective use of Tailwind CSS utility classes to create a professional and user-friendly interface.

#  Features

* Responsive and mobile-friendly design
* Modern landing page with an engaging hero section
* Dedicated Products page showcasing indoor and outdoor plants
* Product search interface
* Informative About section
* Best Seller showcase
* Customer Reviews section
* Contact page with a responsive enquiry form
* Newsletter subscription section
* Clean and intuitive user interface
* Smooth hover effects and responsive layouts

# Technologies Used

* HTML5
* Tailwind CSS (CDN)
